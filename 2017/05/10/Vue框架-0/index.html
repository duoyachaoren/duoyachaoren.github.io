Vue框架

Share

Vue:渐进式框架（SPA）为单页应用程序提供驱动；
主要优点：快速；
主要缺点：因为主要由json文件编写，数据直接交互，不利于搜索引擎查询；
Vue的核心库只关注视图层，它不仅易于上手，还便于与第三方库或既有项目整合。
Vue框架与其他框架的不同之处在于它是直接绑定DOM的数据，操作也是针对数据的，所有元素都是响应式的。
Vue.js 的核心是一个允许采用简洁的模板语法来声明式的将数据渲染进 DOM：
两个大括号包裹想要在浏览器显示的Vue数据
var app = new Vue({
el: ‘#app’,
data: {
message: ‘Hello Vue!’
}
}) 这是声明Vue，（{}）里包裹着一些名值对，el 作用的盒子或区域；data 数据；methous 方法（里面是一些函数）；

Vue的一些指令语句：
属性绑定 v-bind：属性=”属性名”；属性可以是class,href,src等
循环语句 v-for：可以绑定数组的数据来渲染一个项目列表
绑定事件监听 v-on：后面可以是click事件或submit数据
实现表单输入和应用状态之间的双向绑定 v-model：





var app6 = new Vue({
el: ‘#app-6’,
data: {
message: ‘Hello Vue!’
}
})
